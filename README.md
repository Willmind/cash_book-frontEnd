# Egg + React 全栈开发记账本---前端页面代码

前端页面采用使用Vite 2.0 + React + ZarmUI 搭建 H5 开发环境


## 项目启动

```
# 进入项目文件夹
cd react-vite-h5

# 安装依赖
npm install

# 项目运行
npm run dev
使用浏览器打开http://localhost:3000/

# 前端项目打包
npm run build
```
